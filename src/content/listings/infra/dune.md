## Getting Started

1. Create a free account at [dune.com](https://dune.com) to start exploring dashboards and writing queries.
2. Browse community dashboards to find pre-built analytics for protocols, tokens, and trends you are interested in.
3. Fork an existing query to customize it, or write your own SQL query against decoded blockchain tables.
4. Create visualizations and dashboards from your queries to share insights with the community.
