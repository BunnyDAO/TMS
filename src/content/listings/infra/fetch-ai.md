## Getting Started

1. Visit [fetch.ai](https://fetch.ai) and explore the AI agent platform and developer tools.
2. Install the uAgents framework: `pip install uagents` to start building autonomous AI agents.
3. Create a simple agent with built-in communication, task execution, and economic transaction capabilities.
4. Deploy your agent to the Fetch.ai network to interact with other agents in the decentralized marketplace.

## Key Features

- **Autonomous AI agents** build agents that can independently discover services, negotiate, and transact on behalf of users.
- **uAgents framework** lightweight Python framework for creating and deploying AI agents with minimal boilerplate.
- **Agent communication** built-in protocols for agents to discover, communicate, and coordinate with each other.
- **ASI Alliance** part of the merged Artificial Superintelligence Alliance with SingularityNET and Ocean Protocol.
- **DeltaV search** AI-powered search and coordination layer that helps users find and interact with registered agents.
- **Agentverse platform** hosted environment for deploying, managing, and monetizing AI agents without infrastructure management.
