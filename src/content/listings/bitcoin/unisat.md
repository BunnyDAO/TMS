## Getting Started

1. Install the UniSat wallet browser extension from [unisat.io](https://unisat.io) for Chrome or compatible browsers.
2. Create a new wallet or import an existing one using your seed phrase or private key.
3. Browse the marketplace to discover, buy, and sell Ordinals inscriptions and BRC-20 tokens.
4. Use the inscription service to inscribe your own images, text, or data directly onto the Bitcoin blockchain.

## Key Features

- **Built-in marketplace** discover, list, buy, and sell Ordinals inscriptions and BRC-20 tokens without leaving the wallet.
- **Inscription service** inscribe images, text, HTML, and other data onto Bitcoin satoshis directly from the interface.
- **BRC-20 token management** deploy, mint, and transfer BRC-20 fungible tokens on Bitcoin.
- **Runes support** manage and trade Runes protocol tokens alongside Ordinals and BRC-20 assets.
- **UTXO management** advanced controls for managing your Bitcoin UTXOs to protect valuable inscribed satoshis.
- **Fractal Bitcoin support** compatible with Fractal Bitcoin for scaling Bitcoin's programmability layer.
