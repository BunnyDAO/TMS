## Key Features

- **Gaming-first Lightning payments** brings real Bitcoin earnings and spending to mobile and PC games.
- **ZBD API for developers** provides simple REST APIs for integrating Lightning payments into any application.
- **Instant micropayments** enables sub-cent Bitcoin transactions perfect for in-game rewards and tipping.
- **Gamertag system** gives users a human-readable Lightning address for easy peer-to-peer payments.
- **SDK and plugins** offers Unity and Unreal Engine integrations for game developers to add Bitcoin rewards.
- **Compliance-ready** built with KYC/AML compliance for regulated markets and app store compatibility.
