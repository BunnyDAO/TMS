## Getting Started

1. Download Electrum from [electrum.org](https://electrum.org) for Windows, macOS, or Linux and verify the GPG signature.
2. Run the installer and choose to create a new wallet â€” write down the 12-word seed phrase and store it securely offline.
3. Set a strong password to encrypt your wallet file on disk for additional security.
4. Receive your first Bitcoin by sharing your receiving address, and customize transaction fees based on urgency.

## Key Features

- **Lightweight client** connects to remote Electrum servers so you never need to download the full 500GB+ Bitcoin blockchain.
- **Multi-signature support** create wallets requiring multiple keys to authorize transactions for enhanced security.
- **Hardware wallet integration** works seamlessly with Trezor, Ledger, and other popular hardware wallets.
- **Custom fee control** set exact transaction fees manually or use the built-in fee estimator based on network conditions.
- **Cold storage support** create air-gapped offline wallets for maximum security of large Bitcoin holdings.
- **Proven track record** trusted by the Bitcoin community since 2011 with consistent development and security audits.
