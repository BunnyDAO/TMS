## Getting Started

1. Set up a Stacks wallet using Leather (formerly Hiro Wallet) or Xverse to interact with the ecosystem.
2. Acquire STX tokens through a supported exchange or bridge to pay for transactions and participate in stacking.
3. Explore Stacks dApps at [stacks.co/explore](https://stacks.co/explore) for DeFi, NFTs, and other Bitcoin applications.
4. Start Stacking (Stacks' version of staking) to earn BTC rewards by locking your STX tokens.

## Key Features

- **Bitcoin-anchored security** every Stacks transaction is settled on the Bitcoin blockchain through Proof of Transfer.
- **Clarity smart contracts** uses a decidable, non-Turing-complete language designed for security and predictability.
- **sBTC integration** enables trustless Bitcoin DeFi by allowing BTC to move in and out of Stacks smart contracts.
- **BTC yield through Stacking** lock STX tokens to earn Bitcoin rewards, providing native BTC yield.
- **Nakamoto upgrade** brings fast block times and Bitcoin finality to Stacks transactions.
- **Growing DeFi ecosystem** supports DEXs, lending protocols, NFT marketplaces, and social applications built on Bitcoin.
