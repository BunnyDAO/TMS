## Getting Started

1. Deploy BTCPay Server using the one-click install on LunaNode, Voltage, or self-host with Docker on your own server.
2. Create a store and connect your Bitcoin wallet by entering your extended public key (xpub) — your keys, your coins.
3. Set up payment methods including on-chain Bitcoin and Lightning Network for instant settlements.
4. Integrate with your e-commerce platform using plugins for WooCommerce, Shopify, or the API for custom integrations.

## Key Features

- **Zero transaction fees** no processing fees ever — BTCPay Server is completely free and open-source software.
- **Self-hosted sovereignty** run on your own infrastructure with full control over your payment data and funds.
- **Lightning Network support** accept instant payments via Lightning alongside traditional on-chain Bitcoin transactions.
- **E-commerce integrations** plugins available for WooCommerce, Shopify, PrestaShop, Drupal, and other major platforms.
- **Point-of-sale terminal** built-in POS interface for brick-and-mortar businesses to accept Bitcoin in person.
- **Invoicing and accounting** generate payment requests, track invoices, and export data for accounting purposes.
