## Getting Started

1. Install MetaMask from [metamask.io](https://metamask.io) as a browser extension (Chrome, Firefox, Brave, Edge) or mobile app.
2. Create a new wallet and securely back up your 12-word Secret Recovery Phrase â€” never share it with anyone.
3. Add ETH to your wallet by purchasing directly through MetaMask or transferring from an exchange.
4. Connect to dApps by clicking the MetaMask prompt when visiting Web3 applications.

## Key Features

- **Universal dApp connector** the standard wallet for connecting to decentralized applications across the Ethereum ecosystem.
- **Multi-network support** works with Ethereum, Arbitrum, Optimism, Polygon, Base, and any EVM-compatible chain.
- **Built-in swap aggregator** find the best token swap rates across multiple DEXs without leaving the wallet.
- **Token detection** automatically discovers and displays ERC-20 tokens and NFTs in your wallet.
- **Hardware wallet integration** connect Ledger and Trezor devices for enhanced security on all transactions.
- **Snaps extensibility** install community-built plugins to add support for non-EVM chains and custom features.
