## Getting Started

1. Install Backpack from [backpack.app](https://backpack.app) as a browser extension or mobile app.
2. Create a new wallet or import an existing one using your recovery phrase.
3. Fund your wallet with SOL or other tokens from the built-in Backpack Exchange or external transfers.

## Key Features

- **Built-in exchange** Backpack Exchange offers spot and perpetual trading directly integrated with the wallet.
- **xNFT standard** programmable NFTs that embed full applications within the wallet for a new dApp paradigm.
- **Multi-chain support** manages assets across Solana, Ethereum, and other supported blockchains.
- **Clean, modern interface** a polished design that makes managing crypto assets intuitive and enjoyable.
- **Developer platform** build and distribute xNFT applications that live inside users' wallets.
- **Mobile and extension** available as both a browser extension and native mobile app for iOS and Android.
