## Getting Started

1. Install Phantom from [phantom.app](https://phantom.app) as a browser extension or mobile app for iOS/Android.
2. Create a new wallet and securely store your secret recovery phrase in a safe offline location.
3. Fund your wallet with SOL from an exchange or receive tokens from another wallet.
4. Connect to Solana dApps by clicking the Phantom prompt on supported websites.

## Key Features

- **Multi-chain support** manages assets on Solana, Ethereum, Polygon, and Bitcoin in a single unified wallet.
- **Built-in swap** trade tokens directly within the wallet using integrated DEX aggregation for best prices.
- **NFT gallery** beautifully displays your NFT collection with metadata, traits, and collection organization.
- **Staking support** stake SOL directly from the wallet to earn rewards without leaving the app.
- **Transaction simulation** previews the effects of each transaction before signing for security and transparency.
- **In-app browser (mobile)** browse and interact with dApps directly within the Phantom mobile app.
