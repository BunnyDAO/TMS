## Getting Started

1. Try Flux online through platforms like Replicate, fal.ai, or Hugging Face Spaces for instant access without setup.
2. Download model weights from Hugging Face to run Flux locally on your own GPU.
3. Install ComfyUI or use the diffusers library to set up a local generation pipeline with Flux models.
4. Choose between Flux Schnell (fast), Flux Dev (balanced), or Flux Pro (highest quality) based on your needs.

## Key Features

- **Multiple model tiers** including Schnell for speed, Dev for balance, and Pro for maximum quality and detail.
- **Open-source availability** with permissive licensing for the Schnell and Dev models, enabling self-hosting and modification.
- **Exceptional image quality** rivaling Midjourney and DALL-E 3 in aesthetic quality and prompt adherence.
- **Strong text rendering** accurately generates readable text within images across various styles and fonts.
- **ComfyUI integration** works seamlessly with popular open-source workflows for advanced image generation pipelines.
- **ControlNet and LoRA support** enables fine-grained control through pose guidance, style transfer, and custom model adaptations.
