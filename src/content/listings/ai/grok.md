## Getting Started

1. Visit [grok.com](https://grok.com) or open Grok in the X (Twitter) app — free tier available with limited usage.
2. Upgrade to X Premium or SuperGrok for higher rate limits, Grok 4 access, and advanced features like DeepSearch.
3. For API access, sign up at [console.x.ai](https://console.x.ai) — the xAI API is compatible with OpenAI's SDK format.
4. Try asking Grok about trending topics to see real-time X integration in action, or use it for coding and analysis tasks.

## Key Features

- **Grok 4** — xAI's most powerful model, achieving frontier-level scores on math, coding, and reasoning benchmarks (SWE-bench, AIME, GPQA).
- **Real-time X/Twitter integration** — Grok can search and analyze live X posts, trending topics, breaking news, and public discourse in real-time.
- **DeepSearch** — An agentic research mode that performs multi-step web searches, synthesizes findings, and produces comprehensive reports with citations.
- **Massive context window** — Process extremely long documents, codebases, and conversations in a single session.
- **Think mode** — Extended reasoning where Grok shows its step-by-step thought process for complex problems, similar to chain-of-thought approaches.
- **Image generation and vision** — Create images with Aurora (xAI's image model) and analyze uploaded images, screenshots, and visual content.
- **Unfiltered personality** — Known for more direct, witty, and less restricted responses compared to other AI assistants.
- **xAI API** — OpenAI-compatible API format making it easy to integrate into existing applications and switch between providers.
