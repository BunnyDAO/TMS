## Getting Started

1. Visit [chat.mistral.ai](https://chat.mistral.ai) to try Mistral's models through the free Le Chat interface.
2. Sign up for API access at [console.mistral.ai](https://console.mistral.ai) to integrate Mistral models into your applications.
3. Download open-weight models from Hugging Face and run them locally with your preferred inference engine.
4. Explore Mistral's fine-tuning API to customize models on your domain-specific data.

## Key Features

- **Exceptional efficiency** delivering strong performance at smaller parameter counts, reducing inference costs significantly.
- **Open-weight model family** including Mistral 7B, Mixtral 8x7B MoE, and larger frontier models.
- **Mixture-of-Experts architecture** in Mixtral models activates subsets of parameters for faster, cheaper inference.
- **Le Chat platform** provides a polished conversational interface with web search and document analysis.
- **Enterprise-ready API** with function calling, JSON mode, and guardrails for production deployments.
- **Strong European AI alternative** with data sovereignty options and GDPR-compliant hosting.
